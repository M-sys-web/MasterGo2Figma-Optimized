# v3：选中画板可视化修复

本版本修复“选中画板”模式下看不到具体选中对象、容易误以为未生效的问题。

## 新增/调整

- “选中画板”模式下，下方列表会显示已选中的顶层画板/图层名称。
- 顶部摘要显示已选中数量。
- 增加“刷新”按钮：如果插件打开后又重新选择了画板，可以点击刷新读取当前选择。
- 开始导出时会把选中的 nodeId 传给主线程，避免 UI 获得焦点后选中状态丢失导致导出无效。
- 没有选中画板/图层时，“开始”按钮会禁用，并显示明确提示。

## 建议用法

1. 在 MasterGo 画布中选中要迁移的顶层画板/图层。
2. 打开 `SendToFigma Optimized v3`。
3. 导出范围选择“选中画板”。
4. 确认下方列表出现对应画板/图层名称。
5. 如未出现，点击“刷新”。
6. 再点击“开始”。
