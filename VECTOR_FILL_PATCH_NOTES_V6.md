# v6 矢量填充修复说明

本版本针对导入后 Fill / 路径 / 图标类组合看起来缺失填充的问题做了进一步修复。

## 修复点

- 放宽小型 SVG fallback 的触发限制。
- 不再因为整个页面节点数较多就跳过 SVG fallback。
- 对「有填充但 vectorNetwork 缺少 regions」的 PEN / VECTOR 节点，导出端会优先使用 SVG fallback。
- SVG fallback 会保留视觉填充，避免导入 Figma 后只剩路径轮廓或空白路径。

## 代价

部分异常矢量/Logo/图标会以 SVG 节点还原，视觉更稳定，但这类局部图层的可编辑性会低于完全拆成原生路径。普通矩形、文本、Frame、列表、卡片仍按原逻辑导入。

## 使用建议

先用「选中画板」导出一个问题画板，在 Figma 空文件中导入测试。
