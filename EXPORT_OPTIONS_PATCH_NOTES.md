# MasterGo2Figma 导出减负选项补丁

本补丁在 SendToFigma Optimized 基础上增加了导出范围与减负选项，用于降低 MasterGo 导出和 Figma 导入时的卡顿概率。

## 新增导出范围

- 勾选页面：按页面列表勾选导出，适合分批迁移。
- 当前页面：只导出当前打开的页面，忽略页面列表勾选。
- 选中画板：只导出当前页面中已选中的顶层画板/图层，适合迁移单个画板。

## 新增减负选项

- 跳过隐藏图层：导出时忽略隐藏图层及其子图层。
- 跳过草稿页：页面名包含“草稿”或 “draft” 时自动跳过。
- 跳过外部控件：页面名包含“外部控件 / controls / external / 控件库”时自动跳过。
- 大图片自动压缩：超过 3MB 的图片会尝试按最大边 2000px 压缩后再写入 zip 或本地中继服务。

## 注意事项

- “大图片自动压缩”会优先保留更小的压缩结果；如果压缩失败或结果更大，会自动保留原图。
- JPG 图片会按 JPEG 质量 0.82 压缩；PNG / WEBP / BIN 会优先转为 PNG 以尽量保留透明度。
- 如果某些透明大图压缩后视觉异常，可以关闭“大图片自动压缩”后重导。
- 选中画板模式依赖 MasterGo 当前选择，如果未选中任何内容，会提示没有可导出的图层。

## 安装

MasterGo 导入：

```text
SendToFigma/manifest.json
```

Figma 导入：

```text
ReceiveFromMasterGo/manifest.json
```

本补丁只改了 MasterGo 导出端；Figma 导入端沿用上一版 Optimized。
