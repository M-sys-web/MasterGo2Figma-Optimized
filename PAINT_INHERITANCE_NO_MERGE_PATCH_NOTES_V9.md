# v9：填充继承但不合并图层

## 背景
v8 通过将外层带填充的矢量容器整体导出为 SVG，解决了颜色丢失问题，但会把内部路径/组压成一个 SVG 节点，导致图层合并。

## 本版修复
- 不再把带填充的外层 `Fill xx` 容器整体转成 SVG。
- 将外层带颜色的 `Fill xx` 容器还原为透明 Frame。
- 将外层的 fills / strokes 传递给内部没有填充的矢量后代。
- 保留内部组、路径、Fill 子图层层级。

## 取舍
此方案优先保留图层结构。如果某些路径本身缺少 Figma 可识别的 vector regions，继承填充后仍可能不显示；这类极端图形只能用 SVG 兜底，但 SVG 兜底必然会牺牲一部分图层结构。
